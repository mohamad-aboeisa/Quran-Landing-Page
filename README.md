# Quran Landing Page

A Quran landing page project built while practicing HTML and CSS. The project focuses on creating a clean, modern, and responsive layout with well-structured sections and smooth user experience across different devices.

Through this project, I practiced responsive design techniques, page layouts, spacing systems, and reusable styling patterns to improve front-end development and UI design skills.

## Technologies Used
- HTML5
- CSS3
- Flexbox
- CSS Grid
- Responsive Design
